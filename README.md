# OIBSIP_web-development_02

Oasis Infobyte level1-Task 2 – Portfolio Website

 ⭐ Objective
The objective of this task is to create a personal Portfolio Website using HTML and CSS that showcases personal information, skills, projects, achievements, and contact details in a professional and attractive manner.


 🛠 Tools Used
- HTML5  
- CSS3  
- Visual Studio Code  
- GitHub  

📝 Steps Performed
1. Planned the structure and layout of the portfolio website.  
2. Created the webpage using HTML.  
3. Styled the website using CSS for a modern and responsive design.  
4. Added different sections such as:
   - Home  
   - About Me  
   - Skills  
   - Projects  
   - Contact Information  
5. Improved the design using colors, fonts, spacing, and alignment.  
6. Tested the website in the browser and fixed layout issues.  
7. Uploaded the project files to GitHub repository.

  🎯 Outcome
A responsive and visually appealing Portfolio Website was successfully developed.  
The website presents personal details, technical skills, and projects in a professional format and improves front-end web development skills.

   📂 Project Files Included
- `index.html` — Main webpage structure  
- `style.css` — Styling and design  
- `README.md` — Documentation for the task
